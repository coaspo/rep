1. Create branch br1:

1. check in (typical use):
       git add *
       git commit -m "message.."
       git push origin {branch}
2. merge:
       git checkout main
       git merge {branch}
       git push origin main
       branch -d  {ranch}  # delete branch
3. squash:
       git checkout main
       git merge {branch}
       git reset origin/main
       git add . --all
       git push
4. update local:
       git fetch origin {branch}
       git pull origin {branch}
