Blacklist time-wasting websites in windows
Answer: Block news.google.com, edit:
     C:\Windows\System32\drivers\etc\hosts 
  and add, for example: 
    127.0.0.1 news.google.com

Blacklist time-wasting websites in linux
Answer: Block news.google.com, edit:
  sudo gedit /etc/hosts   
        add line:  127.0.0.1 news.google.com  then restart

Spoof MAC address
   Linux: Install (select yes):  sudo apt-get install macchanger   
      Find interface:  ip addr     (should see for example: 2: enp2s0)
      Change MAC addr (replace enp2s0 with actual interface):  sudo macchanger -r enp2s0
      Change MAC after resume from sleep, add command to: /lib/systemd/system-sleep/hdparm
   Windows : run: devmgmt.msc
       Select: >Network adapters
               >Realtek PCIe GBE Family Controller
               >Rt-click >Properties
               >Network Address 
       For value add:
         AABBCCDDEEFF    
     Verify change; run: 
         ipconfig /all
       should have:
         Ethernet adapter Ethernet:
         Physical Address. . . . : AA-BB-CC-DD-EE-FF
