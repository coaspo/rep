Static Website ============== 'w' is a static web site project. It contains usefull/interesting links and/or info. .. contents:: Contents: Test the website ---------------- 1. Open Terminal window 2. Set current dirrectory to parrent directory of ``w``. 3. Run ``python3 -m http.server 8080`` If port is being used, kill the process: sudo kill -9 `sudo lsof -t -i:8080` 4.
