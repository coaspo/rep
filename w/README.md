Static Website
==============
'w' is a static web site project.
It contains useful/interesting links/info.
See [./help.html](help.html) for more details.

.. contents:: Contents:


Test
----
Right click and run [./tests/run_web_site.py](run_web_site.p) 

The script runs:
  fuser -k 8080/tcp
  python3 -m http.server 8080
Site is on a local server;
  http://localhost:8080/w/
