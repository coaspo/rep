Quizzer user doc

    Application can help you study by quickly
creating a multiple choice quiz.
    For example, you select or type "python" in the "topic"
combo box and in the left panel, type the marked text:
    
    /this a one-question quiz  
    ? Select correct statement(s)
    +a, b = b, a
    +[print(x) for x in a_list if x > 10]
    -[if x > 10 print(x) for x in a_list]
    =best features in python
    
where the markers / ? + - =  indicate a
quiz-comment / question / answer / question-comment line.
respectively. With this input application creates a one-question quiz
with 2 correct choices, 1 incorrect choice, and a question comment.
    When all questions are answered, the application displays the score.
Application navigates through a saved set of quizzes and always starts
with the last active quiz.

Recommendations:

  • Make quiz file names more prescriptive by renaming
    quiz files in quz/quizzes. The file name format is
    <topic>-<description>.<id-number>.json
    For example, rename python-.4.json
    to python-classes and OOD quiz.4.json
    The quiz topic python and id-number 4 were not changed.
  • Prevent accidental deletes - make files in quz/quizzes Read-Only
  • Correct/add answer choices by editing the marked text and
    clicking Update
