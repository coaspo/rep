Application is to be used to help study.
It creates a quiz based questions/answers in marked text
with the markers ? + - = ,for example:
    
    ? Select correct statement(s)
    +a, b = b, a
    -[print(x) for x in a_list if x > 10]
    -[if x > 10 print(x) for x in a_list]
    =best features in python
    
that the user enters.  ? + - =  marks the question/answer/comment lines.
After one takes the quiz the application displays the score.

Recommendations:

  • Customize quiz file names; adjust QUIZ_FILE_PFX in quz\config_quizzer.json
  • Correct quizzes - change marked text and click Update
  • Rename quiz files quz/quizzes. using the format:
    <text>.<number>.json - for example:
    python-classes-quiz.4.json
    where 4 is unique in all file names.
    Application loops through a sorted file name.
  • Prevent accidental deletes - make files in quz/quizzes Read-Only
