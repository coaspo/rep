quiz-comment / question / answer / question-comment line.
respectively. With this input, application creates a one-question quiz
with 2 correct answers, 1 incorrect answer, and a comment.

	Application navigates through a saved set of quizzes and always starts
with the last active quiz. It ignores spaces in fill-in answers.

Recommendations:

  • Make quiz file names more prescriptive by renaming

    For example, rename python-.4.json
    to "python-classes and OOD quiz.4.json"
    The quiz topic python can also be changed.
