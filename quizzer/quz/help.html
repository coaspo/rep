Quizzer user doc

Application can help you study by quickly
creating a multiple choice quiz.
For example, you type the marked text:
    
    /this a one-question quiz  
    ? Select correct statement(s)
    +a, b = b, a
    +[print(x) for x in a_list if x > 10]
    -[if x > 10 print(x) for x in a_list]
    =best features in python
    
where the markers / ? + - =  indicate a
quiz-comment/question/answer/question-comment line.
The application creates a multiple choice choice quiz 
with one question with 2/1 correct/incorrect answers,
and a comment.
After you answer all questions, the application displays the score.
You may stop the application and finish a quiz later.

Recommendations:

  • Correct quizzes - change marked text and click Update
  • Make quiz file name more prescriptive -
    rename quiz files quz/quizzes. using the format:
    <topic>-<description>.<id-number>.json
    For example, rename python-.4.json
    to python-classes and OOD quiz.4.json
    The id-number 4 and topic python could but were not changed.
    Application navigates through the sorted file names.
  • Prevent accidental deletes - make files in quz/quizzes Read-Only
