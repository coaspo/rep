Application creates a quiz based on marked text that the user enters.
For example: 
? Select the true statement below
+F=ma
-F=m+a
-F=ma^2
The user may then take the quiz. Application it starts with the question t hat it was closed.

Recommendations:

  • Correct quizzes - change marked text and click Update
  • Rename quiz files ltrans/squiz_files. using the
    format: %lt;text%gt;.%lt;number>.json - for example: infix-functions-quiz.4.json. the number must be unique. Application loops through a sorted file name.
  • Prevent accidental deletes - make files in ltrans/squiz_files Read-Only
