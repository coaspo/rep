Application translates one word at a time.
One can infer and save a more accurate translation.

Key features:

  • Modify any text and add any comments before clicking <Save>
  • Customize combobox default languages by editing ltrans/config_trans.json
  • Adjust translations by editing dictionary files in ltrans/dictionaries
  • For a few languages such as Russian and Greek, use transliteration
    for a rough pronunciation of source or destination words.
    Transliteration converts non-latin characters to latin characters.
